Hackathon March 2019

Specs


Project Premise:

The theme of the hackathon is “code review” - we are asking that groups come up with a solution to post code snippets and allow people to comment and give guidance/feedback

- we will provide a collection of user stories including front end and back end tasks to complete this project

- at the beginning of this hackathon we will split into small groups (you don’t have to know anyone coming in but are welcome to group with people you know if you’d like!)

- your group will have the ability to choose your stack. We will have some recommended boilerplates to start from like vue, react, node, and ruby, but you can decide if you want to incorporate something different if you’d like, for example an api like github

- we will hack through the day and do a retro to talk about what we’ve learned through the day

User Stories

  1. As a developer, I can post a code snippet
  2. As a developer, I can see all the code snippets
  3. As a contributor, I can comment on code snippets
  4. As a developer, I can include my name in my code
    5. snippet
  5. As a contributor, I can include my name with my comment
  6. As a site visitor, I can filter the different kinds of code snippets via language
  7. As a contributor, I can filter to all the code snippets I've made comments on
  8. As a site visitor, there is persistence in the code snippets
  9. As a developer, I can edit or delete my code snippets
  10. As a contributor, I can edit or delete my comments
